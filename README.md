Sistema de Renta de Autos en Solana

Este proyecto consiste en la adaptación de un CRUD básico desarrollado originalmente como una Biblioteca en la blockchain de Solana, el cual fue modificado para convertirse en un sistema de renta de autos.

El programa fue desarrollado utilizando Rust y el framework Anchor Framework dentro del entorno de Solana Playground.

Esta adaptación se realizó como parte del proceso de aprendizaje y práctica para la certificación de desarrollo en Solana, con el objetivo de comprender cómo funcionan los programas on-chain, cuentas, PDAs y transacciones dentro de la blockchain.
